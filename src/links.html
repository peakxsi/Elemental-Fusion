Корисні посилання

Далі також надані корисні посилання для бллижчого ознайомлення з кожною із професій:

Веб-дизайнер
Creative Cloud Інтегрована колекція програм для дизайну та творчості, включаючи Photoshop, Illustrator та інші інструменти
Sketch Професійний інструмент для дизайну інтерфейсів, який дозволяє створювати макети та прототипи веб-сторінок
Figma Онлайн-інструмент для дизайну та спільної роботи над проектами, що дозволяє створювати векторні макети та прототипи
Веб-розробник
MDN Web Docs Відома платформа, що надає документацію та ресурси для веб-розробників, включаючи інформацію про HTML, CSS та JavaScript
W3Schools Онлайн-платформа з навчання веб-технологіям, яка містить практичні приклади, уроки та довідкові матеріали
Stack Overflow Спільнота програмістів, де можна задавати питання, ділитися знаннями та шукати відповіді на технічні питання
Тестувальник
Ministry of Testing Організація, що надає навчальні матеріали, курси та інструменти для професійного тестування програмного забезпечення
Test Automation University Онлайн-университет, який пропонує безкоштовні курси з автоматизації тестування
Software Testing Help Веб-ресурс, що містить статті, поради та ресурси для тестувальників програмного забезпечення
Дана-аналітик
Kaggle Платформа для змагань та співпраці в галузі науки про дані, де можна знайти набори даних, розв'язувати завдання та долучатися до спільноти
DataCamp Онлайн-платформа для навчання аналізу даних та програмування
Towards Data Science Публікації та статті про аналіз даних та машинне навчання, які можна знайти на платформі Medium
